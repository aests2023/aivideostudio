# 구글 스프레드시트 연동 설정 가이드

본 문서는 **AI 영상공방**의 사용자 문의 신청 폼과 관리자 대시보드를 실제 구글 스프레드시트와 실시간 연동하기 위한 설정 단계를 안내합니다.

---

## 1단계: 구글 스프레드시트 생성 및 열(Header) 설정

1. [구글 스프레드시트](https://docs.google.com/spreadsheets)로 이동하여 **새 스프레드시트**를 생성합니다.
2. 첫 번째 행(1행)에 A열부터 H열까지 아래 항목명을 정확히 입력합니다. (항목명이 일치하지 않으면 데이터가 잘못 들어갈 수 있습니다.)

   | 열 (Column) | 항목명 | 설명 |
   | :--- | :--- | :--- |
   | **A열 (A1)** | `ID` | 고유 일련번호 (자동 생성됨) |
   | **B열 (B1)** | `신청일시` | 문의 신청한 날짜와 시간 (자동 입력됨) |
   | **C열 (C1)** | `신청자` | 고객 이름 또는 단체명 |
   | **D열 (D1)** | `연락처` | 고객 연락처 |
   | **E열 (E1)** | `이메일` | 고객 이메일 주소 |
   | **F열 (F1)** | `신청 서비스` | 신청한 서비스 구분코드 (`ad`, `web` 등) |
   | **G열 (G1)** | `상태` | 상담 처리 상태 (`wait` - 상담대기, `ing` - 상담중, `done` - 상담완료) |
   | **H열 (H1)** | `상세내용` | 고객이 작성한 상세 요구사항 |

---

## 2단계: 구글 앱스 스크립트(Apps Script) 복사 및 붙여넣기

1. 스프레드시트 상단 메뉴에서 **[확장 프로그램] > [Apps Script]**를 클릭합니다.
2. 기본으로 생성된 `코드.gs` 파일의 내용(`function myFunction() { ... }`)을 **전부 지웁니다.**
3. 본 프로젝트 폴더에 생성된 [GoogleAppsScript.js](file:///c:/Users/impact/Desktop/안티그라피티IDE/main/GoogleAppsScript.js) 파일의 전체 소스코드를 복사하여 Apps Script 편집기에 붙여넣습니다.
4. 상단의 **저장 버튼 (디스켓 모양 아이콘)**을 누르거나 `Ctrl + S`를 눌러 저장합니다.

---

## 3단계: 웹앱(Web App) 배포 (매우 중요)

스프레드시트 외부(웹사이트 브라우저)에서 해당 스크립트에 접근할 수 있도록 웹 API로 배포해야 합니다.

1. Apps Script 화면 우측 상단의 **[배포] > [새 배포]**를 클릭합니다.
2. 새 배포 창 왼쪽 상단의 **톱니바퀴 아이콘(유형 선택)**을 누르고 **[웹앱]**을 선택합니다.
3. 배포 설정을 아래와 같이 입력합니다:
   - **설명**: `AI 영상공방 API` (자유롭게 입력 가능)
   - **웹앱을 실행할 사용자**: `나 (본인의 Gmail 계정)`
   - **액세스할 수 있는 사용자**: **`모든 사용자` (Anyone)**
     - *주의*: `모든 사용자(Anyone)`로 설정해야 웹사이트를 방문하는 익명의 사용자들이 문의를 등록할 수 있고 대시보드에서 로그인 없이 통신할 수 있습니다.
4. **[배포]** 버튼을 클릭합니다.
5. 배포 도중 구글 계정 액세스 승인을 요구하는 팝업이 나타납니다.
   - **[액세스 승인]**을 클릭하고 구글 계정을 선택합니다.
   - "Google에서 이 앱을 검증하지 않았습니다" 경고 창이 뜨면 좌측 하단의 **[고급]**을 클릭한 뒤, **`AI 영상공방 API(으)로 이동(안전하지 않음)`**을 클릭합니다.
   - 마지막으로 **[허용]**을 클릭합니다.
6. 배포가 완료되면 화면에 **웹앱 URL**이 생성됩니다. 이 URL을 **복사**합니다.
   - *올바른 URL 형식 예시*: `https://script.google.com/macros/s/AKfycb.../exec`

## 6단계: 디스코드 웹훅 알림 설정 (실시간 알림 추가)

회원가입이 발생하거나 상담 신청이 접수되었을 때 디코 채널로 실시간 알림을 받기 위한 설정입니다.

1. **디스코드 서버 및 채널 설정**:
   - 알림을 받을 디스코드 서버를 생성하거나 선택합니다.
   - 알림 전용 텍스트 채널(예: `#알림-봇`)을 선택하고, 채널 이름 옆의 톱니바퀴 아이콘(**채널 편집**)을 클릭합니다.
2. **웹훅 생성 및 URL 복사**:
   - 좌측 메뉴에서 **[연동]** ➔ **[웹훅]**을 클릭합니다.
   - **[웹훅 만들기]** 또는 **[새 웹훅]** 버튼을 클릭합니다.
   - 생성된 웹훅의 이름을 설정하고(예: `AI 영상공방 알림`), 하단의 **[웹훅 URL 복사]** 버튼을 눌러 주소를 클립보드에 복사합니다.
3. **Apps Script에 웹훅 URL 입력**:
   - **Apps Script** 편집기로 돌아가 [GoogleAppsScript.js](file:///c:/Users/gsvlib/Desktop/%EB%82%B4%EC%82%AC%EB%AC%BC%ED%95%A8/main/main/GoogleAppsScript.js)의 34번째 줄 부근을 확인합니다.
   - `var DISCORD_WEBHOOK_URL = 'YOUR_DISCORD_WEBHOOK_URL_HERE';` 부분의 `'YOUR_DISCORD_WEBHOOK_URL_HERE'` 자리에 복사해 둔 웹훅 URL을 붙여넣고 저장합니다.
4. **웹앱 재배포**:
   - 코드를 수정한 후에는 **[배포] > [배포 관리]**를 눌러 기존 버전을 편집하여 새 버전으로 업데이트(재배포)해 주어야 스프레드시트에 변경 사항이 즉시 적용됩니다.
